# Roadmap

Tessera is built in layers. The first release focuses on local deterministic navigation; later releases add richer impact analysis and runtime-aware context.

## v0.1

- Local SQLite graph
- TypeScript, JavaScript, and Python extraction
- CLI query surface
- MCP stdio server
- Definitions, references, outlines, expansion, impact

## v0.2

- Incremental indexing
- Better TypeScript import/export resolution
- More precise method and class member qualification
- Repository benchmark harness
- Go and Rust parser support

## v0.3

- Basic hallucination validator
- Minimal test selection prototype
- Semantic git history experiments
- HTTP/SSE MCP transport

## Later

- Runtime trace ingestion
- OpenTelemetry and error telemetry integrations
- ADR and architecture decision memory
- Token-budgeted query planner
